File tree is a built-in component provided by [Starlight](https://starlight.astro.build/components/file-tree/).

```mdx live
import { FileTree } from "~/components"

<FileTree>
- src
  - docs
    - style-guide
      - components
        - **file-tree.mdx**
</FileTree>
```
