Predefined detection entries are Cloudflare-managed detections for specific types of sensitive content. You can review these entries from the **Predefined** view in **Detection entries**.

You can add any predefined detection entry directly to a custom DLP profile. Use the following reference to review all predefined detection entries currently supported by Cloudflare DLP.
