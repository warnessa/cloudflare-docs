---
title: New ResponseTimeMs field in Gateway DNS Logpush dataset
description: The Gateway DNS Logpush dataset now includes a ResponseTimeMs field for tracking DNS request response times.
date: 2026-04-06
product: Gateway
---

Cloudflare has added a new field to the [Gateway DNS](/logs/logpush/logpush-job/datasets/account/gateway_dns/#responsetimems) Logpush dataset:

- **ResponseTimeMs**: Total response time of the DNS request in milliseconds.

For the complete field definitions, refer to [Gateway DNS dataset](/logs/logpush/logpush-job/datasets/account/gateway_dns/).
