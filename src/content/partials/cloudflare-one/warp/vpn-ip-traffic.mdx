Perform these steps in the [Cloudflare dashboard](https://dash.cloudflare.com/) under **Zero Trust** > **Team & Resources** > **Devices** > **Device profiles** > **General profiles**.

1. Set your [Split Tunnels mode](/cloudflare-one/team-and-resources/devices/cloudflare-one-client/configure/route-traffic/split-tunnels/#change-split-tunnels-mode) to **Exclude IPs and domains**.
2. [Add the following entries](/cloudflare-one/team-and-resources/devices/cloudflare-one-client/configure/route-traffic/split-tunnels/#add-a-route) to your Split Tunnel Exclude list:

   * Private IP address range exposed by your third-party VPN client. For example,
     | Selector | Value |
     | -------- | ----- |
     | IP Address   | `172.16.0.0/12` |
     |
   * Server that your third-party VPN client connects to. For example,
     | Selector | Value |
     | -------- | ----- |
     | Domain   | `*.cvpn-endpoint-xxxxx.prod.clientvpn.us-west-2.amazonaws.com` |
